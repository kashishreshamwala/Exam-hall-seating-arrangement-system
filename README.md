# Digital Examination Seating Allocation System

A web-based system to automate and manage examination seating in educational institutions. Built with **Streamlit**, the system provides smart seating allocation, classroom management, and real-time visualization of student seat assignments.

## Features

- **Role-based access**: Admin, Staff, and Student portals.
- **Automated seating allocation** based on student registration numbers, subjects, and classroom capacities.
- **Interactive classroom visualizer**: View and manage seat arrangements visually.
- **Upload & manage student data**: Staff can upload Excel files with student registrations.
- **Real-time lookup**: Students can view their assigned seats and exam hall information.
- **Admin management**: Add, update, or delete classrooms and seating arrangements.
- **Export functionality**: Download seating arrangements as CSV files.

---

## Technologies Used

- **Python 3.x**
- **Streamlit** for web interface
- **Pandas** for data manipulation
- **Matplotlib** for classroom visualization
- **Firebase Realtime Database** for storing seating data
- **JSON** for classroom configurations
